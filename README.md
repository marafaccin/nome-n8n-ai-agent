# nome-n8n-ai-agent

Este projeto é um workflow criado no n8n que:

- Recebe mensagens do usuário
- Gera respostas com IA (OpenRouter)
- Salva dados no Google Sheets
- Coleta feedback do usuário

## Como usar

1. Importe o arquivo JSON no n8n
2. Configure suas credenciais (Google, API, etc.)
3. Execute o workflow

## Tecnologias
- n8n
- OpenRouter
- Google Sheets
